## 👋 Hi 人，你来了~

我是一名 **生物信息分析工程师**，主要从事  
**多组学数据分析、单细胞转录组分析以及生物信息分析工具与报告系统的开发**。

我曾为 **科研机构、医院及生物医药企业** 提供生物信息分析与计算支持，
